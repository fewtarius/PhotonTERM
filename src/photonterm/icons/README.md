Icons - PhotonTERM v2 logo by fewtarius, released under the GPL.

Master SVG: photonterm.svg (1024x1024, works at all sizes)

Generate all icons from repo root:
  sh src/photonterm/icons/generate-unix.sh   # Linux PNGs
  sh src/photonterm/icons/generate.sh        # macOS ICNS
  sh src/photonterm/icons/generate-win.sh    # Windows ICO

Requires: rsvg-convert (librsvg), ImageMagick (magick), iconutil (macOS)
