# PhotonTERM Icon Requirements

All icons are generated from a single SVG master:

| File | Notes |
|------|-------|
| `src/photonterm/icons/photonterm.svg` | 1024×1024, rounded rect design, works at all sizes |

Generate with the scripts in `src/photonterm/icons/`:

```sh
sh src/photonterm/icons/generate.sh         # macOS ICNS (macOS only, needs iconutil)
sh src/photonterm/icons/generate-unix.sh    # Linux hicolor PNGs
sh src/photonterm/icons/generate-win.sh     # Windows ICO
```

Requires: `rsvg-convert` (librsvg2-bin / `brew install librsvg`), `magick` (ImageMagick 7+), `iconutil` (Xcode CLI tools, macOS only).
